# Code of Conduct

## Our Standard

This project aims to be useful, technically serious, and respectful.

Examples of behavior that contribute to a positive environment include:

- being respectful in technical disagreement
- focusing on facts, reproducibility, and code
- giving actionable feedback
- accepting correction when evidence is clear
- keeping discussion relevant to the project

Examples of unacceptable behavior include:

- personal attacks
- harassment or discriminatory language
- bad-faith argumentation
- deliberate hostility
- publishing private information without permission
- disruptive behavior that prevents productive collaboration

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that violate this Code of Conduct.

Repeated or severe violations may lead to temporary or permanent exclusion from project spaces.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the project maintainer through the platform where this repository is hosted.

## Interpretation

When in doubt, optimize for professionalism, clarity, and respect.
