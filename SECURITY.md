# Security Policy

## Supported Versions

This project is currently developed on the default branch. No separate long-term support branches are maintained at this time.

## Reporting a Vulnerability

Please do not report security issues in public issues.

Instead:

- use a private reporting mechanism offered by the hosting platform, if available
- otherwise contact the maintainer privately through the platform where this repository is hosted

Please include:

- a clear description of the issue
- affected versions or commit range, if known
- reproduction steps
- impact assessment
- any suggested mitigation

## Response Expectations

This is a small open source project and formal response SLAs are not guaranteed, but reasonable efforts will be made to review credible reports and address confirmed issues.
